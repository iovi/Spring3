Веб-приложение Spring3 выводит captcha-картинку с зашифрованным в ней текстом, проверяет предполагаемую разгадку текста, введенную пользователем.

Сборка и запуск

Приложение можно запустить с помощью команды "java -jar Spring3-0.1.0.jar" (файл Spring3-0.1.0.jar находится в директории Spring3\target, необходимо перейти в нее для запуска данной командой). Требуется установленная Java 8 на сервере приложения.
При необходимости можно пересобрать исполняемый файл Spring3-0.1.0.jar с помощью Apache Maven (запустить "mvn clean install" в директории Spring3/ - при этом также будут выполнены тесты приложения).



Описание решения

1. Вывод captcha
Приложение создает и выводит captcha-картинку в ответ на http GET-запрос на адрес http://hostname:8080/captcha, где hostname - имя или IP-адрес сервера, на котором приложение запущено.
На captcha-картинке зашифрован 6-символьный текст, в котором символами могут быть латинские буквы верхнего и нижнего регистра (последующая проверка - регистрозависимая) и цифры. 
В ответе GET-запроса содержатся заголовки:
	captcha-id 	- идентификатор captcha в формате UUID
 	captcha-text 	- текст, который зашифрован на картинке
На выведенной странице содержится поле ввода, в которое пользователь может ввести предполагаемый текст и проверить его по нажатию "Check" (будет отправлен POST-запрос описаный в п.2)

2. Проверка captcha
Приложение принимает http POST-запросы по адресу http://hostname:8080/captcha/check с параметрами id и text, где id - идентификатор captcha (см п.1), text - предполагаемая разгадка.
Разгадка считается верной при выполнении следующих условий:
* Текст из параметра должен совпадать с зашфрованным для указанного id captcha 
* Для одной captcha можно пройти успешную проверку только 1 раз
* Выполнить успешную проверку можно только в течение 1 минуты с момента создания captcha
Если разгадка верна, код ответа на запрос будет равен 200, выведется страница с подтверждением, что текст - корректный. При несоблюдении какого-то из условий, ответ будет иметь код 422, на странице будет выведен соответствующая информация.
